# Self Evaluation

<div class="bg-[#f6f8fa] border border-[#e5e7eb] rounded-lg p-6 my-8">
  <p class="text-[17px] leading-7 text-[#1a1f36]">
Within the first quarter in a new role, I ask each team member to do a self-evaluation using the Level Competencies.  We then review the self-evaluation and discuss any necessary steps to help them grow their skills.
  </p>
</div>

## Assessment Areas

### Technical Skills
- Design craft proficiency
- Tool expertise
- Technical knowledge
- Process understanding

### Soft Skills
- Communication
- Collaboration
- Leadership
- Problem-solving

### Impact
- Project contributions
- Team influence
- Business understanding
- Innovation

## Evaluation Process

1. Review your level competencies
2. Rate yourself in each area
3. Gather concrete examples
4. Identify growth opportunities
5. Create action plans

## Using the Framework

### Step 1: Self-Assessment
- Review each competency
- Rate your current level
- Provide specific examples
- Note areas for improvement

### Step 2: Gap Analysis
- Compare with level expectations
- Identify skill gaps
- Prioritize development areas
- Set realistic goals

### Step 3: Action Planning
- Create specific action items
- Set timelines
- Define success metrics
- Identify resources needed

## Best Practices

- Be honest and objective
- Use specific examples
- Focus on growth opportunities
- Seek feedback from others
- Update regularly
