# Junior Designer (IC1) Competencies

## Core Skills

### Design Craft
- Basic understanding of design principles
- Proficiency with design tools
- Ability to create simple user interfaces
- Understanding of typography and color theory

### Technical Skills
- Basic HTML/CSS knowledge
- Understanding of responsive design
- Familiarity with design systems
- Version control basics

### Collaboration
- Clear communication
- Receptive to feedback
- Works well with guidance
- Team-oriented mindset

## Expected Behaviors

### Ownership
- Takes responsibility for assigned tasks
- Asks questions when unclear
- Meets deadlines
- Maintains organized files

### Growth Mindset
- Eager to learn
- Seeks feedback
- Applies learnings
- Shows initiative

### Communication
- Clear status updates
- Active in team meetings
- Documents work
- Asks for help when needed
